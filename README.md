# Laravel Blog API

## Setup

```bash
composer install
```

Create .env file. Config app and database.

```
APP_NAME=
APP_ENV=
APP_KEY=
APP_DEBUG=
APP_URL=

DB_CONNECTION=
DB_HOST=
DB_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
```

```bash
php artisan key:generate
```

```bash
php artisan migrate --seed
```
